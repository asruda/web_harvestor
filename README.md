# 网页数据抓取工具 (Web Data Crawler)

基于 Python + PyQt6 + QWebEngineView 的可视化网页数据抓取工具。

## 功能特性

- 🖥️ 图形化界面操作，无需编写代码
- 🌐 支持手动登录和Cookie会话保持
- 📊 智能表格数据提取和字段映射
- 📄 支持翻页抓取和链接跟踪
- 💾 多格式导出：Excel、CSV、JSON、文本
- ⚙️ 灵活的抓取策略配置
- 📦 单文件绿色版，解压即用

## 系统要求

- Windows 10 及以上版本 (64位)
- 内存: 建议 4GB 以上
- 磁盘空间: 至少 500MB
- 无需安装 Python 环境

## 快速开始

### 开发环境

```bash
# 安装依赖
pip install -e .

# 运行程序
python main.py
```

## 项目结构

```
web-data-crawler/
├── src/                    # 源代码目录
│   ├── ui/                # 用户界面模块
│   ├── browser/           # 浏览器控制模块
│   ├── crawler/           # 数据抓取引擎
│   ├── database/          # 数据库模型
│   └── utils/             # 工具函数
├── config/                # 配置文件目录
├── data/                  # 数据存储目录
├── main.py                # 程序入口
└── pyproject.toml         # 项目配置
```

## 使用说明

1. 创建网站配置
2. 配置页面抓取规则
3. 选择抓取策略
4. 开始抓取数据
5. 导出结果文件

详细使用教程请参考用户手册。
