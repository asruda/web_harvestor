# 网页数据抓取工具 (Web Data Crawler)

拉取数据

## 使用说明

暂时只支持中src/const/crawl.py中定义的网站

## 功能特性

- 🖥️ 图形化界面操作，无需编写代码
- 🌐 支持手动登录和Cookie会话保持
- 📄 支持翻页抓取和链接跟踪
- 💾 多格式导出：Excel、CSV、JSON
- 📦 单文件夹绿色版，解压即用

## 系统要求

- Windows 10 及以上版本 (64位)
- 内存: 建议 4GB 以上
- 磁盘空间: 至少 500MB
- 无需安装 Python 环境

## 快速开始

### 开发环境

```bash
# 运行程序
uv run main.py
```
